# 🚇 深圳地铁实时客流模拟系统

一个基于原生JavaScript、Canvas和Web数据可视化技术构建的前端项目，专注于展示深圳地铁线路与客流数据。

## ✨ 项目简介

这是一个模拟展示深圳地铁实时客流的单页面应用(SPA)。通过可视化手段，直观呈现了深圳主要地铁线路的站点、换乘信息及模拟的实时客流数据。

**核心价值**：本项目专注于**原生技术栈**的深入实践，旨在夯实前端基础，避免依赖框架黑箱。

## 🔧 技术栈

### 核心技术
- **HTML5** - 语义化标签与Canvas绘图
- **CSS3** - Flex/Grid布局、渐变背景、动画效果
- **JavaScript ES6+** - 模块化、异步编程、事件处理

### 关键技术特性
- **ES6 Modules** - 模块化开发与依赖管理
- **Canvas 2D API** - 热力图可视化渲染
- **Fetch API** - 异步数据加载
- **事件委托与防抖** - 性能优化实践

## 🗺️ 核心功能

### 1. 线路可视化展示
- 完整展示深圳地铁1、2、3、5、11号线
- 采用官方配色方案，视觉识别度高
- 点击切换线路，动态更新站点列表

### 2. 智能交互查询
- **实时站点搜索**：支持模糊查询，带有防抖优化(300ms)
- **跨线路搜索**：全局搜索显示所属线路信息
- **动态高亮**：点击线路按钮高亮显示对应站点

### 3. 数据可视化展示
- **实时客流热力图**：模拟数据可视化展示
- **动态时钟**：页面实时时钟，每秒更新
- **客流统计**：站点客流数据动态生成

### 4. 用户体验优化
- **响应式设计**：完美适配桌面端和移动端
- **平滑动画**：CSS3过渡效果提升交互体验
- **性能优化**：事件委托和防抖技术应用

## 📁 项目结构
```
shen-zhen-demo/
```

## 💡 技术收获

### 工程化实践
- **模块化开发**：将复杂功能拆分为独立可维护的模块（dataLoader.js、lineList.js、stationList.js）
- **状态管理**：掌握通过`currentSelectedLine`管理应用状态的原理
- **错误边界**：实现完善的错误处理机制，保证应用稳定性

### 核心技术深入
- **异步编程**：熟练运用`async/await`处理数据加载，提升代码可读性
- **DOM操作**：精通`document.createElement`、`appendChild`、`innerHTML`等API进行动态渲染
- **事件机制**：深入理解事件监听、回调函数的协同工作原理

### 性能优化实践
- **防抖函数**：在搜索功能中实现防抖优化，避免频繁函数调用
- **事件委托**：应用高效的事件处理机制提升性能
- **Canvas绘图**：掌握2D图形渲染原理和性能优化技巧
